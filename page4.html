E-Note

L’objectif du projet est de proposer une application web pour la gestion des notes du
département (informatique). Elle doit être capable de prendre en entrée les fichiers Excel des
modules, matières, étudiants, notes et les stocker dans la base de données. Elle doit offrir par
la suite la possibilité de parcourir les notes des étudiants par matière, UE, promotion et Année.
L'application doit être hautement sécurisée afin de protéger les données des étudiants
